# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is a data-focused project containing historical email communication records. It is in an early stage with no build system, package manager, or source code yet.

## Project Structure

- `data/` — Contains datasets (e.g., `tikor-success-2010-done.csv`, a ~100MB CSV of Spokane County email metadata from 2010 with columns: Date, Size, From, To)
- `src/` — Source code (currently empty)
- `pages/` — Pages (currently empty)
- `cache/` — Cache directory (currently empty)

## Environment

- Python execution is pre-authorized in `.claude/settings.local.json`
- Not a git repository
